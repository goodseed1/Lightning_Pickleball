번개 테니스(Lightning Tennis) - PROJECT_BLUEPRINT.md (v3)
⓪ Pre-Login & Onboarding: 앱 최초 진입 흐름 (개편)
사용자가 앱을 처음 만나 자신의 여정을 시작하는 가장 중요한 단계입니다.

1. 언어 선택 페이지
   페이지 제목: Select Language / 언어를 선택하세요
   내용: [ English ] 와 [ 한국어 (Korean) ] 두 개의 큰 버튼.
2. 기본 온보딩 (프로필 생성)
   페이지 제목: Welcome! / 환영합니다!
   내용:
   1단계: 계정 생성: 소셜 로그인, 이메일 가입, 서비스 이용약관 및 개인정보 처리방침 동의.
   2단계: 핵심 프로필: 프로필 사진, 닉네임, 성별, LTR 등급.
   3단계: 매칭 정보: 소통 언어(다중선택), 최대 이동 가능 거리.
   4단계: 권한 동의: 위치 정보 및 푸시 알림 허용.
3. AI 컨시어지 온보딩 (신설 - 핵심 경험)
   진입 시점: 사용자가 기본 프로필 생성을 마치고 '🏠 피드' 탭에 최초로 진입했을 때.
   UX 흐름:
   환영 및 목표 질문: AI 챗봇이 사용자의 닉네임을 부르며 환영하고, "오늘 무엇을 가장 먼저 해볼까요?" 라고 질문합니다.
   7대 핵심 목표 제시: 챗봇 대화창에 아래 7가지 목표를 선택할 수 있는 빠른 응답(Quick Reply) 버튼이 표시됩니다.
   🎾 플레이하기 (번개 모임/매치 찾기)
   🙌 모임 주최하기 (번개 모임/매치 호스트)
   🛡️ 클럽 가입하기 (테니스 클럽 찾기)
   👑 클럽 만들기 (테니스 클럽 생성)
   🤝 파트너 찾기 (테니스 파트너 검색)
   🎓 레슨/코치 찾기 (테니스 코치 검색)
   🛠️ 라켓/장비 서비스 (스트링, 수리, 중고 장터)
   실시간 가이드 시작: 사용자가 목표를 선택하면, 챗봇이 해당 목표에 맞는 최적의 화면으로 자동으로 이동시키고, 작은 오버레이(Overlay) 형태로 화면에 남아 단계별로 다음 행동을 안내합니다.
   작업 완료 및 다음 제안: 사용자가 첫 번째 목표를 달성하면(예: 클럽 가입 신청 완료), 챗봇이 축하 메시지와 함께 다음 활동을 자연스럽게 제안하며 지속적인 참여를 유도합니다.
   ① Main App: 주요 탭 바 구조
   로그인 후 앱의 핵심을 이루는 5개의 탭 메뉴입니다.
   아이콘 탭 이름 설명
   🏠 피드 (Feed) 친구와 클럽의 최신 소식을 보는 소셜 허브
   🔍 탐색 (Discover) 이벤트, 플레이어, 클럽 및 서비스를 찾는 탐색 기능
   🎾 생성 (Create) 매치 또는 모임을 즉시 주최하는 핵심 버튼
   🛡️ 내 클럽 (My Clubs) 가입한 클럽 활동을 모아보는 공간
   👤 내 활동 (My Activities) 나의 정보, 통계, 친구, 설정을 관리하는 공간
   ② 각 탭별 상세 페이지 구성
   페이지 제목: Activity Feed
   내용:
   피드 목록: 친구나 가입한 클럽의 활동이 시간순으로 표시됩니다. (예: "Ben님이 Alba님과의 번개 매치에서 승리했습니다", "🏆 Sunday Club의 8월 리그 우승자는 Clara님입니다! 축하해주세요!")
   페이지 제목: Discover
   내용:
   상단 탭 메뉴: [ 🎾 이벤트 ], [ 플레이어 ], [ 클럽 ], [ 🎓 코치 ], [ 🛠️ 서비스 ]
   '이벤트' 탭:
   필터링 버튼: [ 전체 ], [ 🏆 매치만 ], [ 😊 모임만 ]
   이벤트 목록:
   번개 매치 카드: 'Ranked' 또는 트로피 아이콘 🏆 표시.
   번개 모임 카드: 'Casual' 또는 스마일 아이콘 😊 표시.
   '플레이어' 탭:
   추천 플레이어 목록: 나와 조건(지역, 실력, 활동 시간)이 비슷한 플레이어들을 추천.
   검색 기능: 닉네임, LTR 등급으로 플레이어 검색.
   플레이어 카드: 각 플레이어의 프로필 사진, 닉네임, LTR, 지역 정보 표시. 클릭 시 상세 프로필로 이동.
   '클럽' 탭 (클럽 디렉토리):
   검색 및 필터: 지역별, 활동 유형별(경쟁 중심, 소셜 중심 등) 클럽 검색.
   클럽 목록: 주변 클럽들이 로고, 이름, 멤버 수, 소개와 함께 표시됨. 클릭 시 클럽 홈 페이지로 이동.
   '🎓 코치' 탭 (신설):
   코치 목록: 지역 코치들의 프로필, 경력, 레슨비 정보.
   코치 프로필 생성: 코치들이 직접 자신의 프로필, 레슨 가능 시간, 연락처를 등록하고 관리할 수 있는 기능.
   '🛠️ 라켓 서비스' 탭 (신설):
   서비스 목록: 지역의 라켓 줄 교체, 중고 라켓 판매 등 서비스를 제공하는 업체 목록.
   서비스 프로필 생성: 서비스 제공자들이 직접 자신의 업체 정보, 가격, 위치 등을 등록하고 관리할 수 있는 기능.
   페이지 제목: 어떤 종류의 모임을 만드시겠어요?
   내용:
   선택 1: 🏆번개매치(RankedMatch)
   설명: "공식 랭킹과 경기 기록에 영향을 주는 경쟁적인 매치입니다."
   → 클릭 시, 스코어 기록 규칙이 명시된 '번개 매치 생성' 페이지로 이동.
   선택 2: 😊번개모임(CasualMeetup)
   설명: "랭킹과 상관없이 랠리 연습이나 가벼운 게임을 즐기는 소셜 모임입니다."
   → 클릭 시, '모임 목적' 필드가 있는 '번개 모임 생성' 페이지로 이동.
   페이지 제목: My Clubs
   내용:
   가입한 클럽 목록 및 '새 클럽 만들기' 버튼.
   *→ 클럽 선택 시 아래의 \*\*'클럽 홈 페이지'*로 이동.
   페이지 제목: [클럽 이름]
   내용:
   상단 탭 메뉴: [ 홈 ], [ ⚙️ 관리 ](관리자에게만 보임), [ 멤버 ], [ 정기모임 ], [ 리그/토너먼트 ], [ 정책 ], [ 게시판 ],
   '홈' 탭 (업데이트): 클럽 공지사항, 다음 이벤트, 그리고 🏆 최근 리그/토너먼트 우승자가 배지와 함께 자동으로 발표되는 섹션.
   '이벤트' 탭: 클럽장이 생성한 모든 이벤트(정기 모임, 번개 모임/매치) 목록. 클럽 회원은 여기서 각 이벤트에 **'참가 신청'**을 할 수 있음.
   '리그/토너먼트' 탭 (업데이트):
   참가자 기능: 현재 진행 중인 리그/토너먼트 목록. 각 항목을 누르면 순위표/대진표를 보고, 자신의 경기 결과를 직접 입력할 수 있음.
   일반 회원 기능: 관리자가 생성한 리그/토너먼트에 **'참가 신청'**을 하는 기능.
   '📜 정책' 탭 (신설): 관리자가 설정한 클럽의 공식 정책과 규칙(에티켓, 회비 규정 등)을 모든 회원이 열람할 수 있는 페이지.
   페이지 제목: 클럽 관리
   내용: 클럽 운영에 필요한 모든 기능을 포함하는 대시보드.
   회원 관리:
   하위 탭 메뉴: [ 가입 신청 ], [ 전체 회원 ], [ 역할 관리 ]
   회비 관리:
   하위 탭 메뉴: [ 회비 설정 ], [ 납부 현황 ], [ 미납자 관리 ]
   이벤트 생성 및 관리:
   정기 모임 설정 (업데이트): 시스템이 매주 자동으로 이벤트를 생성하면, 회원들이 '이벤트' 탭에서 직접 참가 신청을 해야 함.
   번개 모임/매치 생성: 클럽 멤버 전용의 일회성 이벤트를 생성.
   리그/토너먼트 관리 (업데이트):
   새 리그/토너먼트 생성, 참가자 관리, 경기 결과 관리.
   우승자 확정 (업데이트): 우승자 확정 즉시 '우승자 배지'가 자동으로 수여되고, 클럽 홈과 앱 전체 피드에 공지됨.
   클럽 정책 관리 (신설): 클럽 규칙, 에티켓 등 정책을 작성하고 수정.
   클럽 정보 수정 (신설): 클럽 로고, 이름, 지역, 소개글 등 기본 정보 수정.
   페이지 제목: My Activities
   내용:
   상단 탭 메뉴: [ 내 프로필 ], [ 🏆 경기 통계 ], [ 📅 내 모임 ], [ 친구 ], [ 설정 ]
   '내 프로필' 탭 (업데이트): 프로필 정보, 리그/토너먼트 우승/준우승 배지 표시.
   '🏆 경기 통계 (Match Stats)' 탭: 번개 매치 결과만 반영하는 경기 기록(승/패), ELO 랭킹 포인트 변화 그래프.
   '📅 내 모임 (My Meetups)' 탭: [ 신청한 모임 ], [ 호스트한 모임 ], [ 지난 활동 기록 ] 하위 탭으로 모든 이벤트 참여 상태 관리.
   '친구' 탭: [ 내 친구 ], [ 받은 요청 ], [ 친구 찾기 ] 하위 탭으로 친구 관계 관리.
   '설정' 탭: 알림, 언어 변경, 고객센터, 로그아웃 등.
   ③ 경기/모임 종료 후 흐름
4. '번개 매치' 종료 후:
   1단계: 스코어 입력 화면: 양측이 경기 스코어를 입력하고 상호 확인.
   2단계: 스포츠맨십 평가 화면: 스코어 입력 완료 후, 서로의 스포츠맨십 점수를 평가.
5. '번개 모임' 종료 후:
   1단계: 스포츠맨십 평가 화면: 스코어 입력 없이, 바로 스포츠맨십 평가 화면으로 이동.
   ④ 전역(Global) 컴포넌트 (업데이트)
   앱의 어느 화면에서나 일관되게 접근 가능하며, 앱 전체의 경험을 향상시키는 핵심 컴포넌트입니다.
   모든 페이지에 떠 있는 플로팅 버튼을 통해 접근 가능한 지능형 AI 비서로, 사용자의 상황에 따라 역할이 변화합니다.
   역할 1: AI 컨시어지 (신규/초기 사용자 대상)
   설명: ⓪ Pre-Login & Onboarding 섹션에 상술된 바와 같이, 신규 사용자의 첫 경험을 책임지는 개인화된 온보딩 가이드 역할을 수행합니다. 사용자가 앱의 방대한 기능 속에서 길을 잃지 않도록 돕습니다.
   역할 2: 실시간 도움말 센터 (모든 사용자 대상)
   설명: 사용자가 앱 사용 중 궁금한 점이 생겼을 때, 언제 어디서든 즉시 해답을 제공하는 24시간 고객센터 역할을 합니다.
   지식: '번개 매치 규칙', '리그 점수 기록법', '일반 테니스 상식' 등 앱의 모든 기능과 테니스 관련 지식을 knowledge_base를 통해 학습합니다.
   역할 3: 자동화된 QA 엔지니어 (백그라운드 기능)
   설명: 사용자와의 대화를 통해 앱의 잠재적인 오류나 UX 문제를 실시간으로 감지하고, 개발팀(관리자)에게 자동으로 보고하는 '자동화된 품질 보증' 역할을 수행합니다.
   기능:
   명시적 오류 보고: 사용자가 "에러", "멈춤" 등을 직접 언급 시, 관련 정보를 수집하여 관리자 대시보드에 '버그 리포트' 티켓을 자동 생성합니다.
   암시적 문제 감지: 사용자가 특정 기능에 대해 반복적으로 질문하는 '이상 징후'를 감지하여, 관리자 대시보드에 '잠재적 UX 문제' 티켓을 자동 생성합니다.

### **번개 테니스(Lightning Tennis) - PROJECT_BLUEPRINT.md (v6.5 - Full Version)**

#### **v6.5 주요 변경 사항 요약**

- **'정기 모임' 기능 완성:** 클럽의 반복적인 사교 모임을 위한 `정기 모임` 탭을 신설하고, 관리자의 편의를 위한 지능형 폼(지난 정보 자동 채우기, 복사 기능 등)을 구현했습니다.
- **'리그/토너먼트' 기능 완성:** 관리자가 리그 또는 토너먼트 방식의 대회를 생성하고, 참가자를 모집하며, 자동 대진표/순위표를 생성하고, 경기 결과를 기록하는 전체 흐름을 완성했습니다.
- **관리자 수동 개입 기능:** 자동화된 대회 흐름에 관리자가 직접 개입할 수 있는 유연한 기능(시드 배정, 결과 수정, 기권 처리, 참가자 제외 등)을 추가했습니다.
- **중앙 생성 허브:** `+ 생성` 탭에 '클럽 만들기' 기능을 추가하여, 모든 사용자가 앱의 핵심 단위를 쉽게 생성할 수 있도록 접근성을 향상시켰습니다.
- **사용자 프로필 강화:** 리그/토너먼트 성과에 따라 '트로피'가 수여되는 `명예의 전당`과 개인의 모든 `경기 기록`을 볼 수 있는 기능을 추가했습니다.
- **고도화된 UX/UI:** `클럽 정보` 화면에 스크롤에 반응하는 '다이나믹 헤더'를 적용하고, 전반적인 디자인 시스템을 통일하여 완성도를 높였습니다.

#### **⓪ Pre-Login & Onboarding: 앱 최초 진입 흐름**

사용자가 앱을 처음 만나 자신의 여정을 시작하는 가장 중요한 단계입니다.

**1. 언어 선택 페이지**

- **페이지 제목:** `Select Language / 언어를 선택하세요`
- **내용:** `[ English ]` 와 `[ 한국어 (Korean) ]` 두 개의 큰 버튼.

**2. 기본 온보딩 (프로필 생성)**

- **페이지 제목:** `Welcome! / 환영합니다!`
- **내용:**
  - **1단계: 계정 생성:** 소셜 로그인, 이메일 가입, 서비스 이용약관 및 개인정보 처리방침 동의.
  - **2단계: 핵심 프로필:** 프로필 사진, 닉네임, 성별, LTR 등급.
  - **3단계: 매칭 정보:** 소통 언어(다중선택), 최대 이동 가능 거리.
  - **4단계: 권한 동의:** 위치 정보 및 푸시 알림 허용.

**3. AI 컨시어지 온보딩 (신설 - 핵심 경험)**

- **진입 시점:** 사용자가 기본 프로필 생성을 마치고 **'🏠 피드'** 탭에 최초로 진입했을 때.
- **UX 흐름:**
  1.  **환영 및 목표 질문:** AI 챗봇이 사용자의 닉네임을 부르며 환영하고, "오늘 무엇을 가장 먼저 해볼까요?" 라고 질문합니다.
  2.  **7대 핵심 목표 제시:** 챗봇 대화창에 7가지 목표를 선택할 수 있는 **빠른 응답(Quick Reply) 버튼**이 표시됩니다.

  3.  **실시간 가이드 시작:** 사용자가 목표를 선택하면, 챗봇이 해당 목표에 맞는 최적의 화면으로 **자동으로 이동**시키고, 작은 오버레이(Overlay) 형태로 화면에 남아 **단계별로 다음 행동을 안내**합니다.

###### **↳ (서브 페이지) 클럽 정보 (Club Detail Screen)**

- **헤더 디자인 (신설):** 스크롤에 반응하는 **'다이나믹 커버 헤더'**.
  - **초기 상태:** 클럽의 커버 이미지가 상단을 채우고, 그 위에 로고, 이름, 관리자/멤버 상태가 표시됩니다. 상단 네비게이션 바는 투명합니다.
  - **스크롤 시:** 커버 이미지가 스크롤되어 올라가고, 네비게이션 바에 배경색과 클럽 이름이 나타나며 헤더 영역이 최소화됩니다.
- **상단 탭 메뉴:** `[ 홈 ]`, `[ 멤버 ]`, `[ 🗓️ 정기 모임 ]`, `[ 🏆 리그/토너먼트 ]`, `[ 게시판 ]`, `[ 채팅 ]`, `[ 📜 정책 ]`, `[ ⚙️ 관리 ]`(관리자)

###### **↳ (서브 페이지) 🗓️ 정기 모임 탭 (관리 기능 통합)**

- **역할:** 클럽의 소셜 모임을 확인하고 참여하며, 관리자가 모든 모임을 직접 관리하는 원스톱 허브.
- **관리자 기능 (신설):**
  - **플로팅 액션 버튼 (+):** 언제든지 새로운 수동 모임을 생성합니다.
  - **모임 카드별 메뉴 (`...`):** 각 모임에 대해 `[✏️ 수정]`, `[🗑️ 삭제]`, `[🔁 복사]` 기능을 제공합니다.
- **지능형 폼 (`모임 생성/수정 화면`):**
  - **스마트 위치:** 클럽의 '홈 코트'가 기본으로 자동 설정됩니다. '다른 코트' 선택 시에만 주소 검색 기능이 활성화됩니다.
  - **지난 정보 자동 채우기:** 코트 수, 코트 번호 등은 가장 최근 모임의 정보를 바탕으로 미리 채워집니다.
  - **날짜/시간 선택:** '생성' 또는 '복사' 모드일 때만 날짜와 시간을 자유롭게 변경할 수 있습니다.

###### **↳ (서브 페이지) 🏆 리그/토너먼트 탭**

- **역할:** 클럽 내부의 모든 경쟁적인 대회를 확인하고 참여하며, 관리자가 대회를 관리하는 공간.
- **빈 상태 (Empty State) UI:**
  - **관리자:** 리그가 없을 경우, 중앙에 `[+ 첫 대회 생성하기]` 버튼이 표시됩니다.
  - **일반 회원:** "진행 중인 리그가 없습니다"라는 안내 메시지가 표시됩니다.
- **관리자 기능 (신설):**
  - **플로팅 액션 버튼 (+):** 리그가 하나 이상 있을 때, 추가 리그를 생성합니다.
  - _→ 각 리그 카드를 탭하면 아래의 **'대회 상세 관리 화면'**으로 이동._

###### **↳ (서브 페이지) 대회 상세 관리 화면 (신설)**

- **역할:** 특정 리그 또는 토너먼트의 모든 것을 제어하는 중앙 대시보드. 대회 방식('리그' 또는 '토너먼트')에 따라 표시되는 UI와 기능이 자동으로 변경됩니다.
- **유연한 관리 기능 (신설):**
  - **참가자 관리:** 최종 참가자를 확정하고, **수동으로 시드(Seed)를 배정**할 수 있습니다.
  - **대회 시작:** `[대진표 추첨]` 또는 `[순위표 생성]` 버튼으로 대회를 공식적으로 시작합니다.
  - **진행 관리:**
    - **경기 결과 수정:** 잘못 입력된 스코어나 승자를 정정합니다.
    - **기권 처리:** 특정 선수를 기권패 처리하고 대진표/순위표에 반영합니다.
    - **일정 변경:** 특정 경기의 날짜와 시간을 수동으로 변경합니다.
    - **참가자 제외:** 리그 참가자를 중도 제외하고, 관련 기록의 처리 방식을 선택합니다 (예: 몰수패 처리).

#### **③ 후속 흐름 (Follow-up Flows)**

- **1. '경기(Match)' 종료 후 (자동화):**
  - **스코어 입력:** 양측이 경기 스코어를 입력하고 상호 확인합니다.
  - **통계 업데이트:** 확정 즉시, 참여한 선수들의 프로필에 있는 승/패 기록과 ELO 랭킹 포인트, 그리고 **`경기 기록`**이 자동으로 업데이트됩니다. (Cloud Function)
- **2. '리그/토너먼트' 종료 후 (자동화):**
  - **순위 확정:** 관리자가 대회의 최종 우승자와 준우승자를 확정합니다.
  - **트로피 수여:** 확정 즉시, 우승/준우승자의 프로필 **`명예의 전당`**에 트로피가 자동으로 수여됩니다. (Cloud Function)
  - **피드 공지:** 클럽 피드에 대회 결과가 자동으로 공지됩니다. (Cloud Function)
- **3. '클럽 가입' 승인 후 (자동화):**
  - **피드 공지:** 클럽 피드에 "새로운 멤버가 가입했습니다!" 라는 환영 게시물이 자동으로 생성됩니다. (Cloud Function)
- **4. '클럽 탈퇴' 성공 후 (자동화):**
  - **상태 동기화:** `ClubContext`의 실시간 리스너가 변경을 감지하여, `내 클럽` 목록과 `탐색` 탭의 클럽 상태가 자동으로 즉시 업데이트됩니다.

#### **⑤ Firestore 데이터 구조 (업데이트)**

- **`tennis_clubs/{clubId}`**:
  - `regularMeetupSettings`: 정기 모임 자동 생성을 위한 규칙. **(신규)**
  - `coverImage`: 클럽 상세 화면의 다이나믹 헤더에 사용될 커버 이미지 URL. **(신규)**

  - `messages/{messageId}` (서브컬렉션): 모임 전용 채팅.
- **`leagues_tournaments/{leagueId}`**: 리그/토너먼트 정보.

  - `trophies/{trophyId}` (서브컬렉션): 명예의 전당 기록. **(신규)**
  - `match_history/{matchId}` (서브컬렉션): 개인 경기 기록. **(신규)**
- **`feed/{feedId}`**:
  - 피드 항목에 `type` 필드를 추가하여 '새 멤버 가입', '리그 시작', '경기 결과' 등 다양한 종류를 구분. **(업데이트)**
