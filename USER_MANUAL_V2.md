> 현재 앱의 **실제 구현된 기능**만을 기술한 공식 문서입니다.
>
> **최종 업데이트**: 2025년 12월 14일

## 📱 앱 개요

**번개 피클볼**는 피클볼인들이 자유롭게 만나고, 경쟁하며, 성장할 수 있는 디지털 커뮤니티 플랫폼입니다.

### 핵심 가치

- **자율성**: 사용자가 직접 경기와 모임을 만들고 운영
- **공정성**: ELO 레이팅 시스템을 통한 객관적 실력 평가
- **투명성**: 모든 경기 기록과 랭킹의 공개

## 1️⃣ 탐색 (Discover) 탭

### 1.1 상단 필터 탭 (6개)

1. **이벤트**: 번개 매치/모임 목록
2. **플레이어**: 주변 피클볼인 검색
3. **클럽**: 피클볼 클럽 찾기
4. **코치**: 레슨 게시판 (게시물 형태)
5. **서비스**: 줄 교체, 중고거래 등 (게시물 형태)

### 1.2 이벤트 탭 상세

- **필터 옵션**: 전체 / 매치만 / 모임만
- **표시 정보**: 이벤트 제목, 일시, 장소, 호스트, 참가 인원
- **거리 기반 정렬**: 사용자 위치 기준 가까운 순

### 1.3 플레이어 탭 상세

- **검색 기준**: 거리, LPR 수준
- **표시 정보**: 닉네임, LPR, 거리, 전적 (승/패)
- **액션**: 프로필 보기, 친구 요청

### 1.4 클럽 탭 상세

- **검색 기준**: 거리, 클럽명
- **표시 정보**: 클럽명, 로고, 멤버 수, 거리
- **액션**: 클럽 상세 보기, 가입 신청

### 1.5 코치 탭 상세 (게시판 형식)

- **누구나 등록 가능**: 로그인한 사용자면 레슨 홍보 가능
- **필수 항목**: 제목, 설명
- **선택 항목**: 일시, 장소, 레슨료, 최대 인원
- **게시 제한**: 1인당 하루 3개, 총 최대 5개
- **연락 방법**: "상담하기" 버튼 → 1:1 채팅방

### 1.6 서비스 탭 상세 (게시판 형식)

- **카테고리**: 줄 교체, 패들 수리, 중고 패들, 중고 장비, 기타
- **필수 항목**: 제목
- **선택 항목**: 설명, 카테고리, 사진(최대 5장), 가격
- **게시 제한**: 1인당 하루 3개, 총 최대 5개
- **연락 방법**: "연락하기" 버튼 → 1:1 채팅방
- **가격 표시**: 작성자 국가 기준 통화 (달러/$, 원/₩)

## 2️⃣ 이벤트 시스템

### 2.1 이벤트 종류

| 종류          | 설명             | ELO 반영  |
| ------------- | ---------------- | --------- |
| **번개 매치** | 공식 랭킹 경기   | ✅ 예     |
| **번개 모임** | 가벼운 소셜 모임 | ❌ 아니오 |

### 2.2 이벤트 생성 (Create 탭)

1. **"새 이벤트 만들기"** 선택
2. **종류 선택**: 매치 또는 모임
3. **형식 선택**: 단식(2명) 또는 복식(4명)
4. **정보 입력**: 날짜, 시간, 장소, 설명
5. **생성 완료**: 호스트로 자동 참가

### 2.3 복식 참가 방식

- **솔로 참가**: 파트너 없이 혼자 신청 (매칭 대기)
- **팀 참가**: 앱 내 친구를 파트너로 지정하여 함께 신청

### 2.4 복식 LPR 허용 범위

- 팀의 평균 LPR이 상대 팀과 **±0.5 범위** 내에 있어야 참가 가능
- 예: 호스트팀 평균 3.5 → 참가팀은 3.0~4.0 허용

### 2.5 경기 종료 후 흐름

1. **스코어 입력** (번개 매치만)
2. **스포츠맨십 평가**: 상대방에게 태그 부여
   - 😊 친절함, 🎯 페어플레이, ⏰ 시간준수, 💪 열정적 등

## 3️⃣ ELO 랭킹 시스템

### 3.1 두 개의 독립된 랭킹

| 랭킹 종류    | 반영 경기          | 상호 영향 |
| ------------ | ------------------ | --------- |
| **전체 ELO** | 공용 번개 매치     | ❌ 없음   |
| **클럽 ELO** | 클럽 리그/토너먼트 | ❌ 없음   |

> **핵심 정책 - "분리 독립" 모델**
>
> 클럽 내부 경기(리그, 토너먼트)는 **클럽 ELO에만** 영향을 주며,
> 사용자의 **전체 ELO 랭킹에는 영향을 주지 않습니다.**
>
> 마찬가지로, 공용 번개 매치 결과는 클럽 ELO에 영향을 주지 않습니다.

### 3.2 ELO 재경기 제한

- **단식**: 동일 상대와 **3개월에 1회**만 ELO 반영
- **복식**: 완전히 동일한 팀 조합만 3개월 제한 적용. 팀 구성원 중 한 명이라도 다르면 기록경기로 인정
- 제한 기간 내 재경기는 **친선 경기**로 진행 (기록만 남고 ELO 변동 없음)

### 3.3 K-Factor (클럽 ELO)

| 경기 유형            | K값 | 특성                   |
| -------------------- | --- | ---------------------- |
| 클럽 리그            | 16  | 안정적, 꾸준한 성장    |
| 클럽 토너먼트 (신규) | 32  | 빠른 레벨 탐색         |
| 클럽 토너먼트 (기존) | 24  | 극적인 변동, 업셋 보상 |

## 4️⃣ 클럽 시스템

### 4.1 클럽 역할

| 역할              | 권한                                 |
| ----------------- | ------------------------------------ |
| **관리자(Admin)** | 모든 권한 (설정, 회원관리, 대회개최) |
| **매니저**        | 일부 관리 권한                       |
| **운영자**        | 회비 관리 권한                       |
| **일반 멤버**     | 참가, 열람                           |

### 4.2 클럽 기능

- **홈**: 클럽 정보, 공지사항
- **멤버**: 회원 목록
- **리그/토너먼트**: 클럽 내 대회
- **채팅**: 클럽 단체 채팅방
- **정기모임**: 반복 일정 모임
- **회비관리**: 월/분기/연 회비 납부 현황 (운영자 이상)

### 4.3 리그/토너먼트 시스템

- **리그**: 라운드 로빈 방식, 순위표 자동 생성
- **토너먼트**: 싱글 엘리미네이션, 대진표 자동 생성
- **참가 신청**: 회원이 직접 신청
- **결과 입력**: 경기 참가자가 직접 스코어 입력
- **승인 방식**: 양측 확인 또는 관리자 승인

## 📋 자주 묻는 질문 (FAQ)

### Q: 복식 경기에 솔로로 참가할 수 있나요?

**A:** 네, 가능합니다. 복식 이벤트에서 "솔로 참가"를 선택하면 파트너 없이 신청할 수 있습니다. 다른 솔로 참가자와 자동으로 매칭됩니다.

### Q: 클럽 경기가 제 전체 ELO에 영향을 주나요?

**A:** 아니요. "분리 독립" 모델에 따라, 클럽 내부 경기(리그, 토너먼트)는 **클럽 ELO에만** 영향을 주며, 전체 ELO 랭킹에는 영향을 주지 않습니다.

### Q: 같은 상대와 여러 번 경기해도 ELO가 반영되나요?

**A:** 동일 상대와는 **3개월에 1회**만 ELO가 반영됩니다. 그 이전에 재경기하면 "친선 경기"로 처리되어 기록은 남지만 ELO 변동은 없습니다.

### Q: 코치/서비스 게시글은 몇 개까지 올릴 수 있나요?

**A:** 도용 방지를 위해 **하루 최대 3개**, **총 최대 5개**(활성 상태)까지 등록할 수 있습니다. 기존 게시글을 삭제하면 새로 등록할 수 있습니다.

### Q: 복식 참가 시 LPR 제한이 있나요?

**A:** 네, 팀의 평균 LPR이 상대 팀과 **±0.5 범위** 내에 있어야 합니다. 예를 들어, 호스트팀 평균이 3.5면 참가팀 평균은 3.0~4.0이어야 합니다.

### Q: 서비스 게시판에서 연락처를 공개할 수 있나요?

**A:** 아니요. 개인정보 보호를 위해 연락처 공개는 불가합니다. 대신 **"연락하기" 버튼**을 통해 앱 내 1:1 채팅으로만 연락할 수 있습니다.
