## 클럽 생성자 (Club Creator) UX Flow

### 🎯 사용자 프로필

**페르소나**: 김테니스 (35세, 테니스 동호회장)

- **목표**: 지역 테니스 동호회를 디지털화하여 체계적으로 운영하고 싶음
- **동기**: 기존 카카오톡 단체방의 한계를 극복하고 전문적인 클럽 관리 원함
- **우려사항**: 기술적 복잡성, 기존 멤버들의 앱 적응도

### 📱 상세 UX 여정

#### Phase 1: 발견 및 온보딩 (Discovery & Onboarding)

```
🔍 문제 인식
"카카오톡으로 30명 관리하기 너무 복잡해..."
↓
📱 앱 다운로드
"Lightning Tennis 앱을 발견"
↓
🌐 언어 선택
[ English ] [ 한국어 (Korean) ] → 한국어 선택
↓
📝 계정 생성 (4단계)
1️⃣ 소셜 로그인 → Apple ID 선택
2️⃣ 핵심 프로필 → 닉네임: "김클럽장", LTR: 4.0
3️⃣ 매칭 정보 → 활동지역: 서울 강남구, 이동거리: 30km
4️⃣ 권한 동의 → 위치정보, 푸시알림 허용
```

**감정 상태**: 기대감 😊 → 약간의 복잡함 😐 → 완료 만족 😌

#### Phase 2: 클럽 생성 과정 (Club Creation)

```
🏠 피드 탭 진입
"비어있는 피드... 뭔가 만들어야겠다"
↓
🛡️ 내 클럽 탭 이동
"가입한 클럽 목록 및 [새 클럽 만들기] 버튼 발견"
↓
📋 클럽 생성 폼 작성
• 클럽명: "강남 테니스 클럽"
• 소개: "매주 토요일 정기 모임을 갖는 강남구 기반 테니스 동호회입니다"
• 로고: 클럽 단체사진 업로드
• 지역: 서울 강남구
• 공개설정: 공개 클럽 ✅
• 코트 주소: 강남구민체육센터 테니스장
• 비용 정보:
  - 가입비: 20,000원
  - 월회비: 30,000원
  - 년회비: 300,000원
• 시설: 조명시설 ✅, 주차장 ✅
• 클럽 규칙: "예의를 지키고 즐겁게 운동하기"
↓
⏰ 정기 모임 설정
• 요일: 토요일
• 시간: 오후 2:00 PM
• 장소: 강남구민체육센터
↓
🎉 클럽 생성 완료
"클럽이 성공적으로 생성되었습니다!"
```

**감정 상태**: 설렘 😍 → 집중 🤔 → 성취감 🎉

#### Phase 3: 초기 클럽 설정 (Initial Club Setup)

```
🏛️ 클럽 홈페이지 자동 진입
"[강남 테니스 클럽] 페이지"
↓
⚙️ 관리자 탭 접근
"클럽 운영에 필요한 모든 기능 대시보드"
↓
📜 클럽 정책 설정
• 에티켓 규정 작성
• 회비 규정 명시
• 참가 취소 정책 수립
↓
💰 회비 관리 설정
• QR코드 업로드 (카카오페이, 토스)
• 납부 마감일: 매월 25일
↓
🎯 첫 번째 이벤트 생성
"번개 모임: 신입 회원 환영회"
• 날짜: 이번 주 토요일
• 시간: 오후 2:00-4:00
• 최대 인원: 20명
• 내용: "기존 멤버와 신규 가입자 간의 친목 도모"
```

**감정 상태**: 책임감 💪 → 체계적 만족 📊 → 기대감 ⭐

#### Phase 4: 회원 관리 단계 (Member Management)

```
📬 가입 신청 알림 수신
"5명의 새로운 가입 신청이 있습니다"
↓
👥 가입 신청 검토
• 프로필 확인: LTR 수준, 활동 지역, 자기소개
• 승인/거절 결정
• 승인 시 환영 메시지 자동 발송
↓
🏆 리그/토너먼트 기획
"9월 클럽 내 복식 토너먼트"
• 참가 신청 기간: 2주
• 대진표 자동 생성
• 우승상품: 클럽 로고 텀블러
↓
📊 회원 활동 모니터링
• 정기 모임 참석률 확인
• 회비 납부 현황 점검
• 미납자 알림 발송
```

**감정 상태**: 뿌듯함 😊 → 책임감 💼 → 성장 실감 📈

#### Phase 5: 클럽 운영 최적화 (Club Optimization)

```
📈 클럽 성장 관리
• 회원 수: 15명 → 35명 증가
• 월 활동: 4회 정기 모임 + 2회 번개 모임
• 만족도: 평균 4.8/5.0
↓
🤝 클럽 간 연합 활동
• 다른 클럽과 친선 경기 기획
• 지역 토너먼트 공동 주최
↓
🎖️ 우승자 배지 수여
"9월 토너먼트 우승자: 이윤수님"
• 우승 배지 자동 수여
• 클럽 홈 및 앱 피드에 공지
• 다음 시즌 시드 자격 부여
```

**감정 상태**: 자부심 😎 → 커뮤니티 리더십 👑 → 지속 발전 의지 🚀

## 클럽 가입자 (Club Member) UX Flow

### 🎯 사용자 프로필

**페르소나**: 박신입 (28세, 테니스 초보자)

- **목표**: 테니스를 배우고 함께 운동할 사람들을 만나고 싶음
- **동기**: 건강 관리 + 새로운 인맥 형성
- **우려사항**: 실력 부족에 대한 두려움, 클럽 분위기 적응

### 📱 상세 UX 여정

#### Phase 1: 클럽 발견 (Club Discovery)

```
🔍 탐색 탭 진입
"Discover 페이지"
↓
🏛️ 클럽 탭 선택
"상단 탭 메뉴: [ 🎾 이벤트 ], [ 플레이어 ], [클럽], [ 🎓 코치 ], [ 🛠️ 라켓 서비스 ]"
↓
🎯 클럽 검색 & 필터링
• 지역: 강남구 인근
• 활동 유형: 소셜 중심
• 실력 수준: 초급자 환영
↓
📋 클럽 목록 탐색
"강남 테니스 클럽"
• 로고: 단체 사진
• 멤버 수: 35명
• 소개: "매주 토요일 정기 모임... 초보자 환영"
• 평점: ⭐⭐⭐⭐⭐ (4.8/5.0)
↓
👆 클럽 상세 정보 확인
"클럽 홈페이지로 이동"
```

**감정 상태**: 호기심 🤔 → 관심 증가 😯 → 긍정적 기대 😊

#### Phase 2: 클럽 탐색 및 정보 수집 (Club Research)

```
🏠 클럽 홈 탭 확인
• 클럽 공지사항 읽기
• 다음 이벤트: "신입 회원 환영회"
• 최근 우승자: 이윤수님 🏆
↓
👥 멤버 탭 확인
• 전체 멤버 프로필 탐색
• LTR 분포: 초급자 40%, 중급자 60%
• 나이대: 20-40대 골고루 분포
↓
📅 이벤트 탭 확인
• 정기 모임: 매주 토요일 2시
• 번개 모임: 월 2-3회
• 참가비: 무료 (회비 납부자)
↓
📜 정책 탭 확인
• 클럽 에티켓: "서로 존중하며 즐겁게"
• 회비: 가입비 2만원, 월회비 3만원
• 참가 취소: 하루 전까지 가능
```

**감정 상태**: 신중함 🤨 → 안정감 😌 → 결정 의지 💪

#### Phase 3: 가입 신청 (Membership Application)

```
✋ 가입 신청 버튼 클릭
"가입 신청서 작성"
↓
📝 자기소개 작성
"안녕하세요! 테니스를 배우고 싶은 직장인입니다.
실력은 부족하지만 열심히 배우겠습니다.
매주 토요일 참석 가능합니다."
↓
⏳ 승인 대기
"가입 신청이 완료되었습니다.
관리자 승인을 기다려주세요."
↓
📬 승인 알림 수신
"축하합니다! 강남 테니스 클럽 가입이 승인되었습니다."
↓
🎉 환영 메시지 확인
"환영합니다! 이번 주 환영회에 참석해보세요."
```

**감정 상태**: 떨림 😰 → 기대감 ⏰ → 기쁨 🎉 → 소속감 🤗

#### Phase 4: 첫 활동 참여 (First Activity)

```
📅 첫 이벤트 참가 신청
"신입 회원 환영회 - 토요일 오후 2시"
• 참가 신청 버튼 클릭
• 자동 승인 (클럽 멤버)
↓
🎾 환영회 당일
• 클럽 홈에서 참석자 명단 확인
• 위치 정보 및 준비물 체크
• 클럽장의 환영 인사
↓
🤝 첫 만남 & 네트워킹
• 기존 멤버들과 인사
• 간단한 실력 테스트
• 파트너 매칭 및 연습 게임
↓
💬 클럽 채팅 참여
"오늘 정말 즐거웠습니다! 다음에도 참석하겠습니다."
↓
😊 스포츠맨십 평가
"모든 멤버에게 5점 만점 평가"
```

**감정 상태**: 긴장 😬 → 환영받는 느낌 🤗 → 즐거움 😄 → 소속감 강화 💕

#### Phase 5: 정기 활동 참여 (Regular Participation)

```
🔄 주간 루틴 확립
매주 화요일: 다음 토요일 정기 모임 참가 신청
매주 토요일: 정기 모임 참석
매월 1회: 번개 모임 참여
↓
📈 실력 향상 실감
• LTR 2.5 → 3.0 성장
• 파트너 매칭 시 선호도 증가
• 베이직 샷 안정성 향상
↓
🏆 첫 토너먼트 참가
"9월 클럽 내 복식 토너먼트"
• 참가 신청
• 대진표 확인
• 첫 경기: 2세트 아깝게 패배
• 경기 결과 직접 입력
↓
👫 클럽 내 친구 형성
• 실력이 비슷한 멤버와 친구 추가
• 개인적 연습 약속 잡기
• 다른 클럽 이벤트 함께 참여
```

**감정 상태**: 안정감 😌 → 성장 만족 📈 → 도전 의식 🔥 → 깊은 유대감 👥

#### Phase 6: 커뮤니티 기여 (Community Contribution)

```
🎯 적극적 참여 활동
• 신입 멤버 멘토링 자원
• 이벤트 기획 의견 제시
• 클럽 SNS 홍보 참여
↓
🏅 첫 수상 경험
"11월 토너먼트 준우승"
• 준우승 배지 획득 🥈
• 프로필에 자동 표시
• 클럽 홈에 공지
• 개인 피드에 자랑 게시
↓
🌟 롤모델 전환
"이제 나도 신입 멤버들을 도와주는 선배"
• 클럽 운영 참여 의사 표현
• 부 운영진 역할 제안 받음
• 다른 초보자들의 상담 역할
```

**감정 상태**: 기여 욕구 💪 → 성취감 🏆 → 리더십 발현 👑 → 클럽 애착 💖

## 상호작용 시나리오

### 시나리오 1: 회비 관리 상황

```
📱 클럽 생성자 시점:
"회비 납부 마감일이 다가왔다"
↓
💰 미납자 관리 탭 확인
• 5명의 미납자 발견
• 개별 납부 독려 알림 발송
• 결제 QR코드 다시 공유

📱 클럽 가입자 시점:
"앗, 회비 납부를 깜빡했다!"
↓
📬 알림 수신 확인
• 친근한 톤의 납부 독려 메시지
• 간편 결제 QR코드 스캔
• 납부 완료 알림 자동 발송
```

### 시나리오 2: 이벤트 참가 관리

```
📱 클럽 생성자 시점:
"번개 모임 인원이 부족하다"
↓
📢 추가 홍보 활동
• 클럽 채팅에 재공지
• 참가 혜택 추가 (간식 제공)
• 참가자 명단 실시간 확인

📱 클럽 가입자 시점:
"재미있어 보이는 이벤트 발견"
↓
🤔 참가 고민
• 이벤트 상세 정보 확인
• 참가자 명단 및 실력 분포 체크
• 일정 조정 후 참가 신청
```

### 시나리오 3: 분쟁 해결

```
🚨 문제 상황:
"토너먼트에서 스코어 분쟁 발생"

📱 클럽 생성자 대응:
↓
⚖️ 중재 개입
• 양측 의견 청취
• 영상/증인 확인
• 클럽 규정에 따른 판정
• 스포츠맨십 교육 실시

📱 클럽 가입자 반응:
↓
🤝 갈등 해결
• 관리자 판정 수용
• 상대방과 화해
• 더 나은 스포츠맨십 다짐
```

### 시나리오 4: 클럽 성장 관리

```
📱 클럽 생성자 고민:
"멤버가 50명을 넘어서 관리가 어렵다"
↓
🔧 운영 체계 개선
• 부 운영진 임명
• 소그룹(레벨별) 운영 도입
• 이벤트 자동화 설정 활용

📱 기존 클럽 가입자:
"클럽이 너무 커져서 친밀감이 줄어든다"
↓
💭 적응 과정
• 소그룹 활동 참여
• 친한 멤버들과의 소모임 형성
• 새로운 역할(멘토) 수용
```

## 🎯 핵심 UX 인사이트

### 클럽 생성자의 성공 요인

1. **직관적인 설정 과정**: 복잡한 관리 기능을 단계별로 안내
2. **자동화 기능**: 정기 이벤트 생성, 배지 수여 등 반복 작업 최소화
3. **실시간 피드백**: 회원 활동, 납부 현황 등 즉시 확인 가능
4. **확장 가능성**: 클럽 규모 성장에 따른 기능 확장
